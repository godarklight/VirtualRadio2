All packets:

VR2 (3 bytes) | byte MESSAGE_TYPE (1 byte) | [PAYLOAD VARIABLE]

Payload:

Heartbeat - 0 bytes
Connection end - 0 bytes to variable, UTF8 encoded string for the disconnect reason
Set VFO - 4 bytes int (Hz)
Set Mode - 1 byte (ModeTypes)
Set Rate - 4 bytes int
Set IQ - 1 byte bool
Data - 2 bytes sequence number, 512 bytes of u8 baseband audio data
